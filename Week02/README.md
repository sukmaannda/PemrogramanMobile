Pemrograman Mobile - Week01 

Pada pertemuan ini saya mempelajari dasar-dasar bahasa pemrograman Dart yang digunakan dalam pengembangan aplikasi mobile, khususnya sebagai bahasa utama pada framework Flutter. Materi diawali dengan pengenalan Dart sebagai bahasa yang berorientasi objek (object-oriented) serta memiliki fitur null safety yang membantu mencegah kesalahan akibat nilai null. Setiap program Dart dijalankan melalui fungsi main() yang berperan sebagai titik awal (entry point) eksekusi program.

Selanjutnya, saya mempelajari konsep variabel sebagai tempat untuk menyimpan data yang dapat digunakan dan diolah selama program berjalan. Variabel dapat dideklarasikan menggunakan var dengan type inference atau dengan menyebutkan tipe data secara eksplisit seperti int, double, String, dan bool. Selain itu, terdapat penggunaan final untuk variabel yang nilainya ditetapkan satu kali saat runtime, serta const untuk nilai tetap yang sudah diketahui saat compile-time.

Materi juga membahas berbagai tipe data yang tersedia di Dart, baik tipe data primitif seperti angka, teks, dan boolean, maupun tipe data non-primitif seperti List untuk menyimpan kumpulan data terurut dan Map untuk menyimpan pasangan key-value. Pemahaman terhadap tipe data ini sangat penting agar data dapat dikelola secara tepat sesuai kebutuhan program.

Selain itu, saya mempelajari berbagai jenis operator yang digunakan dalam pengolahan data dan penyusunan logika program. Operator aritmatika digunakan untuk melakukan perhitungan matematika, operator relasional untuk membandingkan nilai, operator logika untuk menentukan kondisi, operator penugasan untuk menetapkan dan memperbarui nilai variabel, serta operator null-aware untuk menangani kemungkinan nilai null dengan lebih aman.

Dari keseluruhan materi, saya memahami bahwa variabel, tipe data, dan operator merupakan fondasi utama dalam pemrograman. Penguasaan konsep-konsep dasar ini sangat penting karena menjadi landasan dalam membangun logika program sebelum mengembangkan aplikasi mobile yang lebih kompleks dan terstruktur.
