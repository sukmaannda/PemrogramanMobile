Pemrograman Mobile - Week01
Pada pertemuan pertama praktikum Pemrograman Mobile, kegiatan diawali dengan proses instalasi Flutter SDK. File Flutter diunduh kemudian diekstrak ke direktori lokal yang mudah diakses. Setelah itu, folder bin ditambahkan ke dalam Environment Variables pada bagian Path agar perintah Flutter dan Dart dapat dijalankan melalui Command Prompt atau terminal.

Proses instalasi selanjutnya dicek menggunakan perintah flutter --version dan flutter doctor untuk memastikan bahwa Flutter telah terpasang dengan benar serta seluruh kebutuhan (dependencies) sistem sudah terpenuhi.

Selain itu, dilakukan juga instalasi Android Studio beserta Android SDK hingga aplikasi berhasil dijalankan dan menampilkan halaman awal sebagai tanda bahwa instalasi telah berhasil.

Sebagai tahap akhir pengujian, dibuat aplikasi sederhana bertuliskan “Hello World” untuk memastikan bahwa konfigurasi Flutter dan Android Studio sudah berjalan dengan baik dan siap digunakan dalam proses pengembangan aplikasi mobile.
